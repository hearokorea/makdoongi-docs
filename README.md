# 막둥이 (Makdoongi)

한국 주식 시장용 **AI 자동 매매 프로그램**.

시장을 분석하고 매매를 결정하는 일은 AI가 맡고, 사용자는 텔레그램으로 결과를
받아보며 필요할 때 개입합니다.

---

## 무엇을 하나요

- 장 중 시세·뉴스·거시 지표를 자동 수집해 매매 기회를 찾습니다.
- 익절·손절·트레일링 스탑 같은 리스크 규칙을 실시간으로 따라갑니다.
- 하루가 끝나면 오늘의 거래를 스스로 되돌아보고 다음 날 판단에 반영합니다.
- 예상치 못한 상황(시세 지연, 체결 오류, 장외 시간 등)은 스스로 감지하고 대응합니다.
- 모든 활동은 텔레그램으로 실시간 보고됩니다.

---

## 누구에게 맞나요

- 한국 주식 시장에서 자동매매를 시도해본 **개인 투자자**
- 매 순간 시세에 매달리기보다 **시스템에 맡기고 싶은 사용자**
- 매매 근거와 과정을 **기록으로 남기고 싶은 사용자**

**적합하지 않은 경우**: 수동 매매 선호, 초단위 차익 거래 전용, 해외 주식 전용.

## 주요 기능

| 분야 | 기능 |
|---|---|
| 🤖 자동 매매 | AI 판단 기반 매수·매도 결정 / 전략 실시간 스크리닝 |
| 🛡 리스크 관리 | 익절·손절·트레일링 스탑 / 일일 손실 상한 / 연속 손실 시 자동 보수화 |
| 🧠 자가 학습 | 일일·주간·월간 복기 / 반복 손실 패턴 자동 발견 / 조기 익절 기회손실 학습 |
| 📱 텔레그램 | 현황·계좌·전략·리스크 조회 / 자유 입력으로 AI와 대화 / 긴급 제어 |
| 🔧 안정성 | 크래시 자동 복구 / 외부 주문(HTS/앱) 자동 동기화 / 유령 포지션 자동 정리 |
| 🔍 자기 감사 | 시스템이 설계대로 동작하고 있는지 스스로 주기적 점검 |
