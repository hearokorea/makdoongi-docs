# 막둥이 (Makdoongi)

한국 주식 시장용 **AI 자동 매매 프로그램**.

시장을 분석하고 매매를 결정하는 일은 AI가 맡고, 사용자는 텔레그램으로 결과를 받아보며 필요할 때 개입합니다.

변경 이력 → [CHANGELOG.md](CHANGELOG.md)

---

## 무엇을 하나요

- 장중 시세·뉴스·거시 지표를 자동 수집해 매매 기회를 찾습니다.
- 익절·손절·트레일링 스탑 같은 리스크 규칙을 실시간으로 따라갑니다.
- 하루가 끝나면 오늘의 거래를 스스로 되돌아보고 다음 날 판단에 반영합니다.
- 예상치 못한 상황(시세 지연, 체결 오류, 장외 시간 등)은 스스로 감지하고 대응합니다.
- 모든 활동은 텔레그램으로 실시간 보고됩니다.

---

## 누구에게 맞나요

- 한국 주식 시장에서 자동매매를 시도해본 **개인 투자자**
- 매 순간 시세에 매달리기보다 **시스템에 맡기고 싶은 사용자**
- 매매 근거와 과정을 **기록으로 남기고 싶은 사용자**

**적합하지 않은 경우**: 수동 매매 선호, 초단위 차익 거래 전용, 해외 주식 전용.

## 주요 기능

### 🤖 자동 매매
- AI가 시장 상황과 보유 종목을 종합해 매수·매도·관망을 결정합니다.
- 여러 매매 전략이 항상 후보 종목을 탐색합니다.
- 텔레그램으로 지시하면 즉시 반영합니다.
- 수수료·슬리피지 비효율을 막기 위해 너무 적은 금액의 매매는 자동으로 회피합니다.

### 🛡 리스크 관리
- 종목별 익절·손절·트레일링 스탑을 자동 적용합니다.
- 하루 손실이 일정 수준에 도달하면 당일 매매를 중단합니다.
- 손실이 반복되면 자동으로 보수적 모드로 전환합니다.
- 장 시간·공휴일·프리마켓을 구분해 다르게 동작합니다.

### 🧠 자가 복기와 학습
- 큰 손실이나 강제 매도 직후 원인을 분석하고 알림으로 제공합니다.
- 일일·주간·월간 복기가 자동 생성·전송됩니다.
- 반복되는 손실 조합을 찾아 다음 판단에서 회피합니다.
- "너무 일찍 익절해서 더 오르는 걸 놓친" 기회손실 케이스를 추적해 익절 기준을 스스로 조정합니다.
- 감지된 문제에 맞춰 운영 기준을 자동 조정하고, 일정 기간 후 효과를 측정해 악화되면 원상 복구합니다.

### 🔍 자기 진단
- 최근 24시간 매매 성과를 기준으로 5단계 자기 판정을 내립니다. (우수 · 양호 · 주의 · 위험 · 회복 중)
- 시스템 자신이 설계대로 동작하고 있는지 주기적으로 스스로 점검합니다. 이상 발견 시 자동으로 수리 큐에 등록하고 알림을 전달합니다.

### 📱 텔레그램 연동
- 모바일에서 현황·계좌·전략·리스크를 바로 확인합니다.
- 자연어로 AI에게 직접 질문하거나 지시할 수 있습니다.
- 자동매매 일시 중단·재개, 긴급 매도를 한 번에 지시할 수 있습니다.
- 중요 알림은 즉시, 반복 알림은 자동으로 빈도를 조정합니다.

### 🔧 안정성과 복구
- 프로그램 재시작 후에도 포지션·전략·학습 상태가 이어집니다.
- HTS·모바일 앱·전화 주문으로 거래해도 자동으로 동기화됩니다.
- 실제 보유가 없는 잔여 기록을 안전하게 정리합니다.
- 체결 확인이 늦게 도착해도 기록이 올바르게 갱신됩니다.
- 증권사 API가 일시 불안정할 때 무리한 재시도 대신 안정될 때까지 자동 대기합니다.

### 💰 운영 비용 관리
- AI 호출 비용 상한이 일·월 단위로 관리됩니다.
- 사용자 명령과 자동 판단을 분리 집계해, 자동 비용이 한도에 도달해도 사용자 대화는 유지됩니다.
- 날짜가 바뀌면 한도가 자동으로 초기화됩니다.

## 텔레그램 화면 예시

**현황 조회**
```
📊 현황 | 2h15m 가동

💰 계좌
총자산 48,120,000 | 평가 6,870,000 | 🔴+120,000

📍 종목 (4/10)
삼성중공업 30주 @31,650 🔴+3.2%
포스코인터내셔널 50주 @79,200 🔴+1.0%
...

🖥 프로그램
가동 2h15m | API 사용 $0.41/5 | 매수AI:활성 | 매도AI:활성
```

**자연어 질문**
```
사용자: 오늘 왜 삼성중공업 안 팔았어?

막둥이:
현재 +3.2% 수익 중이며, 기술적 지표가 상승 지속을 시사합니다.
익절 기준까지 여유가 있어 트레일링 스탑으로 따라가는 중입니다.
장마감 전 급락 시 즉시 매도하도록 감시 중입니다.
```

**자기 진단 알림**
```
🔴 시스템 자기진단: STRUGGLING
· 24h 매매 15건 (매수 5/매도 3)
· 승률 45% / 손절율 20%
· 추세: 회복 / 구조 재검토 권장
```
