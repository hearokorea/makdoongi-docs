# 막둥이 (Makdoongi)

한국 주식 시장을 위한 **AI 기반 개인 자동 매매 프로그램**.

막둥이는 AI 가 시장을 분석하고 판단해서 자동으로 매매를 수행합니다.
사용자는 텔레그램으로 현황을 확인하고 명령을 내릴 수 있습니다.

## ✨ 주요 기능

### 🤖 AI 자율 매매
- 시장 분석부터 매수·매도까지 AI 가 자율 결정
- 매일 매매 결과를 스스로 복기해 판단을 점진 개선
- 상황에 맞는 AI 모델 자동 선택으로 비용 최적화

### 🛡 실시간 리스크 관리
- 강제 손절 · 손절 · 익절 · 트레일링 스탑 자동 실행
- 자투리 매매 차단 + 기존 자투리 포지션 본전 회복 시 자동 청산
- 연속 손실·매도 실패 감지 시 자동 보수화 (신규 매수 차단 등)

### 📱 텔레그램 연동
- 어디서나 현황 확인과 긴급 제어 가능
- 자유 입력으로 AI 에게 직접 질문/명령
- 모바일 한 화면에 컴팩트하게 핵심 정보 표시

### 🧠 자가 복기·학습·자율 개선
- **즉시 복기** — 큰 손실 발생 직후 1초 내 원인 분석 + 개선 제안
- **일일/주간/월간 복기** — 자동 실행, 텔레그램 리포트
- **자동 학습 (손실)** — 반복되는 손실 패턴을 스스로 발견해 다음 판단에서 회피
- **자동 학습 (기회손실)** — 너무 일찍 익절한 케이스를 매도 후 30분 추적 → 패턴 반복 시 익절 기준 자동 상향
- **자기 진단** — 24시간 기준 자기 상태를 5단계로 판정해 매시간 알림
- **자율 조정** — 감지된 문제에 맞춰 운영 기준을 자동 조정 (안전 범위 내, 3일 효과 측정 + 악화 시 자동 원복)

### 🔧 안정성과 복원력
- 크래시·프로그램 종료 후 부팅 시 자동 복구
- 미완료 작업·포지션 자동 재개, 유령 포지션 자동 정리
- 체결 확인이 지연된 주문 자동 교차검증 + 뒤늦게 도착한 체결 정보로 자동 기록 복구
- HTS·모바일·전화 주문 등 외부 거래 자동 인식 (자기 주문과 혼동 차단)
- 한국 공휴일·프리마켓 자동 인식 — 장외 시간 경보 오탐 차단
- 재시작 후에도 자기 진단·경보 기록·학습 상태 모두 보존

### 💰 비용 의식형 운영
- AI 일일 비용 한도 자동 차단 (자정 자동 리셋)
- 사용자 텔레그램 명령은 한도와 별도 관리 — 대화 단절 방지
- 고비용 AI 수리 건은 자동 처리 대신 사용자가 몰아서 승인
