# 기능 안내

막둥이의 주요 기능을 카테고리별로 정리합니다.
세부 구현·수치는 운영자 관리 범위이므로 이 문서에서는 다루지 않습니다.

---

## 1. 자동 매매

- **AI 주도 판단** — 시장 상황과 보유 종목을 종합해 매수·매도·관망 결정을 내립니다.
- **전략 자동 스크리닝** — 여러 개의 매매 전략이 항상 후보 종목을 탐색합니다.
- **사용자 의도 수용** — 텔레그램으로 지시하면 즉시 반영합니다.
- **건당 주문 최소 금액 보호** — 수수료·슬리피지 비효율을 막기 위해 너무 적은
  금액의 매매는 자동으로 회피합니다.

## 2. 리스크 관리

- **익절 / 손절 / 강제 손절** — 종목별로 자동 설정됩니다.
- **트레일링 스탑** — 수익이 나면 고점을 따라가며 보호선을 올립니다.
- **일일 손실 상한** — 하루 손실이 일정 수준에 도달하면 당일 매매를 중단합니다.
- **연속 손실 보호** — 손실이 반복되면 자동으로 보수적 모드로 전환합니다.
- **장외 시간 보호** — 장 시간·공휴일·프리마켓 구분해 적용합니다.

## 3. 자가 복기와 학습

- **즉시 복기** — 큰 손실이나 강제 매도 직후 원인을 분석하고 알림으로 제공합니다.
- **일일·주간·월간 복기** — 자동 생성·전송됩니다.
- **손실 패턴 자동 발견** — 반복되는 손실 조합을 찾아 다음 판단에서 회피합니다.
- **기회손실 학습** — "너무 일찍 익절해서 더 오르는 걸 놓친" 케이스를 추적해
  익절 기준을 스스로 조정합니다.
- **자율 조정** — 감지된 문제에 맞춰 운영 기준을 자동 조정합니다. 일정 기간 후
  효과를 측정하고 악화되면 원상 복구합니다.

## 4. 자기 진단

- **실시간 상태 판정** — 최근 24시간 매매 성과를 기준으로 5단계 자기 판정을 내립니다.
  (우수 · 양호 · 주의 · 위험 · 회복 중)
- **자가 감사** — 자기 자신의 시스템이 설계대로 동작하고 있는지 주기적으로 점검합니다.
  이상 발견 시 자동으로 수리 큐에 등록하고 알림으로 전달합니다.

## 5. 텔레그램 연동

- **어디서나 조회** — 모바일에서 현황·계좌·전략·리스크를 바로 확인합니다.
- **자유 입력** — "오늘 성과 어때", "삼성전자 어떻게 보나" 같은 자연어로
  AI에게 직접 질문할 수 있습니다.
- **긴급 제어** — 자동매매 일시 중단·재개, 긴급 매도를 한 번에 지시할 수 있습니다.
- **알림 우선순위** — 중요 알림은 즉시, 반복 알림은 자동으로 빈도 조정됩니다.

## 6. 안정성과 복구

- **자동 재시작 복구** — 프로그램 재시작 후에도 포지션·전략·학습 상태가 이어집니다.
- **외부 주문 인식** — HTS·모바일 앱·전화 주문으로 거래해도 자동으로 동기화됩니다.
- **유령 포지션 자동 정리** — 실제 보유가 없는 잔여 기록을 안전하게 정리합니다.
- **지연 체결 자동 복구** — 체결 확인이 늦게 도착해도 기록이 올바르게 갱신됩니다.
- **데이터 조회 불안정 대응** — 증권사 API가 일시 불안정할 때 무리한 재시도 대신
  안정될 때까지 자동으로 대기합니다.

## 7. 운영 비용 관리

- **AI 호출 비용 상한** — 하루·한 달 사용량 한도가 정해집니다.
- **사용자 명령 별도 관리** — 자동 판단과 사용자 질문을 분리 집계해, 자동 비용이
  한도에 도달해도 사용자 대화는 유지됩니다.
- **자정 자동 리셋** — 날짜가 바뀌면 한도도 자동으로 초기화됩니다.

## 8. 규칙 기반 자동 처리

- AI 호출 전에 명확한 규칙으로 처리 가능한 경우는 규칙 엔진이 먼저 판단합니다.
  이를 통해 AI 호출 비용을 줄이고, 판단 속도를 높입니다.

## 사용 예시

### 간단한 조회

> 원장님: `/포지션`
>
> 막둥이:
> ```
> 📍 보유 4/10
> 삼성중공업 30주 @31,650 🔴+3.2%
> 포스코인터내셔널 50주 @79,200 🔴+1.0%
> 플루토스 1000주 @521 🔴+6.9%
> 씨아이에스 30주 @16,720 🔵-0.8%
> ```

### 자유 입력

> 원장님: 오늘 왜 삼성중공업 안 팔았어?
>
> 막둥이:
> ```
> 💬 현재 +3.2% 수익 중이며, 기술적 지표(MACD +127, RSI 60)가
> 상승 지속을 시사합니다. 익절 기준인 +20%까지 여유가 있어
> 트레일링 스탑으로 따라가는 중입니다.
> 장마감 전 급락 시 즉시 매도하도록 감시 중입니다.
> ```

### 이상 감지 알림

> ```
> 🔴 시스템 자기진단: STRUGGLING
> · 24h 매매 8건 (매수 3/매도 5)
> · 승률 20% / 손절율 40%
> · 추세: 악화 — 보수 모드 자동 전환됨
> ```
