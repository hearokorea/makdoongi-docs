# 변경 이력

사용자가 체감하는 새 기능·변화를 버전/날짜 역순으로 정리합니다.
세부 사용법은 [MANUAL.md](MANUAL.md) 참조.

## 2026-04-22

### 새로운 기능
- **자기 진단** — 시스템이 24시간 기준 자기 상태를 5단계 (`excellent` / `good` / `cautious` / `struggling` / `recovering`) 로 판정. 매시간 텔레그램 알림.
- **자율 튜닝** — 연속 손실·매도 실패·오탐 감지 시 운영 파라미터를 자동 조정. 3일 후 효과 측정하고 악화 시 자동 원복.
- **자동 패턴 발견** — 반복되는 손실 조합을 시스템이 스스로 발견 + AI 판단에 자동 반영.
- **자투리 매매 근절** — 건당 주문 금액 하한(기본 50만원) + 기존 자투리 포지션 본전 회복 시 자동 청산.
- **유령 포지션 자동 정리** — 실제 보유 없는 포지션 3회 시도 후 자동 제거 + 알림 1회 (기존 5회 반복 스팸 차단).

### 새로운 명령
- `/selfstate` — 24시간 자기 진단 조회
- `/tune` — 자율 튜닝 현황 (`/tune history`, `/tune params`)

### 개선
- `/health` 에 학습·자가인식 지표 통합 표시
- 자기 주문이 외부 거래로 오분류되던 문제 해결 (체결 지연 대응)
- 포지션 태그 `[장기/AI-초단타]` 같은 불일치 자동 교정
- 일일 결산 금액 표시 정확도 향상 (TR 실패 시 캐시 활용)
- 무거래 / 루프 정체 / 중장기 정체 경보의 맥락 인식 (점심시간·장마감·재시작 직후 등 정당한 상황은 자동 skip)
- 재시작 시 미완료 사용자 승인 액션 자동 재개 (24시간 이내)

## 2026-04-21

### 새로운 기능
- **시스템 자가 복기 (5분 주기)** — 매매 없이 AI 비용만 발생하는 이상 상황을 자동 감지해 알림 + 수리 큐 등록
- **오탐 감사 (10분 주기)** — 시스템이 발령한 경보가 사후 잘못된 것으로 드러나는 패턴 감사
- **즉시 복기** — 강제 손절/-3% 이상 손실 직후 1초 내 패턴 자동 분류 + 개선 제안
- **과거·현재·미래 통합 판단** — 손실 패턴 DB + 연속 손실 + 1주 전 대비 + 경제 캘린더 + 실적/공시 예정을 매 AI 루프에 자동 주입
- **항상성 컨트롤러** — 포트폴리오·브로커 상태 이탈 시 자동 보수 모드 전환

### 새로운 명령
- `/health` — 시스템 전체 건강도
- `/repairs` — 수리 큐 조회
- `/reload` — 코드 핫 리로드

### 개선
- 재시작 시 포지션 안전 보호 (자동 청산 기본 OFF, 옵트인)
- 외부 거래(HTS·모바일·전화 주문) 자동 감지 정확도 향상
- 프로그램 지연 체결 자동 기록 (네트워크 지연으로 늦게 체결된 주문 인식)
- 모바일 알림 컴팩트 포맷 (한 화면에 핵심 정보 표시)
