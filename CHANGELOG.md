# 변경 이력

막둥이에 추가된 주요 기능의 **등장 시점** 만 간단히 정리합니다.
세부 구현·수치는 공개 대상이 아닙니다.

---

## 2026-03 — 첫 릴리스 · 기본 골격

- AI 기반 자동 매매 시스템 시작
- 스케줄 모드 (개장 전 브리핑 / 장 중 / 장 마감 후)
- 텔레그램 조회·명령·자유 입력
- 워치독 (크래시 자동 감지 + 자동 복구)
- 일일 브리핑 자동 생성
- 브로커 연결 자동 재시도
- 과다 매수 안전 장치
- 실시간 매도 감시 (트레일링 스탑 포함)
- 하이브리드 AI 매도 판단 (익절 시점에 AI가 "보유 vs 매도" 결정)
- 재시작 없이 변경 반영되는 **핫 리로드** 체계 도입

## 2026-04 전반 — 중장기 전략 + 비용 최적화

- 중장기·초단타 듀얼 루프 체계
- 뉴스 이벤트 실시간 감지 → 즉시 매매 판단
- 24시간 인텔리전스 수집 (장외 시간 포함)
- AI 호출 비용 극한 최적화 (규칙 엔진 + 사전 계산)
- 24시간 운영 안정화 (자동 복구 전 범위 강화)

## 2026-04 중반 — 복기·학습·기술 부채 체계

- 즉시 복기 / 일일·주간·월간 복기 자동화
- 반복 손실 패턴 자동 발견
- 기술 부채 레지스트리 (코드 개선 대기열)
- 시스템 자가 점검 (주기적 건강성 체크)

## 2026-04 후반 — 자가 인식과 자율 진화

- 자기 상태 진단 (5단계 자기 판정)
- 자율 튜닝 (안전 범위 내 자동 조정 + 효과 측정 + 악화 시 자동 원복)
- 항상성 컨트롤러 (이탈 시 기본값 복귀)
- 외부 거래 자동 인식·동기화 (HTS·앱·전화 주문)
- 자투리 매매 자동 차단
- 유령 포지션 자동 정리
- 오탐 감사 (경보 품질 자동 관리)
- 로그·메시지 아카이브 (모든 기록 30일 보존)
- 축 간 이벤트 버스 (축끼리 실시간 소통)

## 2026-04-22 — 유기체 7단계 완성

- 기억 · 배우기 · 깨우치기 · 통찰 · 자가수정 · 항상성 · 자기진단
  7단계가 서로 맞물려 끊김 없이 순환
- 텔레그램 사용자 명령은 AI 비용 한도와 별도 집계

## 2026-04-23 — 자가 감사 체계 + 치명 버그 일괄 정비

### 주요 기능 추가
- **시스템 원칙 자가 감사** — 막둥이가 스스로 "설계대로 동작 중인가"를
  주기적으로 자문. 위반 발견 시 자동 수리 큐 등록 + 알림.
- **기회손실 학습** — 조기 익절 후 가격 추적으로 익절 기준 자동 보정.
- **pending 주문 자동 검증** — 체결 확인이 지연되는 주문을 백그라운드에서
  교차 검증해 누락 없이 기록.
- **거래 기록 자동 보정** — 프로그램 시작 시 지난 24시간 모호한 기록을
  자동 복구.
- **공휴일·프리마켓 자동 인식** — 장외 시간 경보 오탐 차단.

### 핵심 버그 일괄 수정
- AI 비용 한도 자정 이후 영구 차단 해소
- 매도 실패 반복 / 유령 포지션 재생성 방지
- 외부 거래 오탐 (자기 주문이 외부로 분류되던 문제)
- 자가진단 승률 0% 집계 버그
- 개장·재시작 직후 루프 정체 경보 오탐
- 중장기 루프 정상 스케줄임에도 경보 발생 문제

### 관측 강화
- 텔레그램 알림 메시지 한글화 · 이모지 일관성 개선
- `/selfstate` 에 시스템 원칙 감사 결과 추가
- 수리 큐 항목 카테고리 한글 표시

### 자기 참조 정합성
- 막둥이가 자기 자신의 코드 무결성도 스스로 점검하도록 확장
- 내부 API 변경을 자동 탐지하는 검증 체계 추가
