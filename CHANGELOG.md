## 2026-04-22 — 유기체 진화 7-Stage + 학습·자가 인식 5단계 100%

### 새로운 기능
- **자기 진단** — 시스템이 24시간 기준 자기 상태를 5단계 (`excellent` / `good` / `cautious` / `struggling` / `recovering`) 로 판정. 매시간 텔레그램 알림.
- **자율 튜닝** — 연속 손실·매도 실패·오탐 감지 시 운영 파라미터를 자동 조정. 3일 후 효과 측정하고 악화 시 자동 원복.
- **자동 패턴 발견** — 반복되는 손실 조합을 시스템이 스스로 발견 + AI 판단에 자동 반영.
- **자투리 매매 근절** — 건당 주문 금액 하한(기본 50만원) + 기존 자투리 포지션 본전 회복 시 자동 청산.
- **유령 포지션 자동 정리** — 실제 보유 없는 포지션 3회 시도 후 자동 제거 + 알림 1회 (기존 5회 반복 스팸 차단).
- **로그·메시지 아카이브** — 모든 기록 30일 보관 (SQLite)
- **항상성 컨트롤러** — 포트폴리오/브로커 상태 이탈 시 자동 보수화
- **Foundation 4 축간 이벤트 버스** — 축간 pull → push 전환

### 새로운 명령
- `/selfstate` — 24시간 자기 진단 조회
- `/tune` — 자율 튜닝 현황 (`/tune history`, `/tune params`)
- `/reload` — 코드 핫 리로드

### 개선
- `/health` 에 학습·자가인식 5단계 지표 통합
- 자기 주문이 외부 거래로 오분류되던 문제 해결 (체결 지연 대응)
- 포지션 태그 `[장기/AI-초단타]` 같은 불일치 자동 교정
- 일일 결산 금액 표시 정확도 향상 (TR 실패 시 캐시 활용)
- 무거래 / 루프 정체 / 중장기 정체 경보의 맥락 인식 (점심시간·장마감·재시작 직후 등 정당한 상황은 자동 skip)
- 재시작 시 미완료 사용자 승인 액션 자동 재개 (24시간 이내)
- 스키마 정합성 자동 관리 (코드 업데이트 시 레거시 데이터 자동 교정)
- **텔레그램 사용자 명령은 AI 비용 한도 bypass** (대화 단절 방지)
- 문서 체계 재정리 — 코드 레포(상세 개발 이력) / 문서 레포(사용자 관점) 분리
