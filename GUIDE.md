# Makdoongi 사용자 가이드

> 처음 사용하시는 분을 위한 쉬운 안내서입니다.

---

## 1. Makdoongi가 뭔가요?

Makdoongi는 **개인 계좌로 한국 주식을 AI가 자동으로 매매해 주는 프로그램**입니다.

- 아침에 프로그램을 켜두면 장이 열리는 동안 알아서 매매합니다.
- 텔레그램 메신저로 프로그램과 대화할 수 있습니다.
- 장이 끝나면 결과를 자동으로 정리해서 알려줍니다.

## 2. 준비물

- **Windows PC** — 집에 놓고 상시 켜둘 수 있는 컴퓨터
- **키움증권 계좌** — OpenAPI+ 사용 신청 필요
- **텔레그램** — 메시지 받을 스마트폰 앱

> 설치나 계정 연동은 운영자가 도와드립니다.

## 3. 시작 흐름

```
  아침                     장중                     저녁
  ────────────────────────────────────────────────────
  ① 프로그램 실행    →   ② 자동 매매 진행   →   ③ 결산 보고
     (또는 자동시작)       (텔레그램 알림)        (자동 전송)
```

1. **프로그램 실행**
   - 바탕화면 아이콘을 더블클릭하면 시작합니다.
   - 실행되면 텔레그램으로 "시작 보고"가 옵니다.

2. **장 시간 동안**
   - AI가 종목을 고르고 매수/매도를 결정합니다.
   - 주요 이벤트(매수 성공, 손절 등)는 텔레그램으로 바로 알려줍니다.
   - 궁금할 때는 `/status`를 보내면 현재 상황을 알려줍니다.

3. **장 마감 후**
   - 하루 거래 내역이 자동으로 정리돼서 옵니다.
   - 프로그램은 지정 시각(기본 15:35)에 자동 종료됩니다.

## 4. 처음 해볼 텔레그램 명령

프로그램이 실행 중일 때 텔레그램에서 아래 명령을 보내보세요.

| 입력 | 받는 내용 |
|---|---|
| `/status` | 지금 상태 (연결, 잔고, 보유 종목) |
| `포지션` | 보유 종목 상세 |
| `성과` | 오늘/최근 매매 결과 |
| `시장` | 현재 시장 분위기 |
| `결산` | 하루 결산 미리보기 |

> 한글 명령도 슬래시 없이 그대로 입력하면 됩니다.

## 5. 안전 장치

막둥이는 여러 안전 장치를 갖추고 있습니다.

- **자동 손절** — 미리 정한 손실 % 에 도달하면 자동으로 팔아줍니다.
- **일일 한도** — 하루 거래 횟수와 금액에 상한을 둡니다.
- **비상 정지** — 위급할 때 텔레그램으로 `/emergency_stop`을 보내면
  모든 매매가 즉시 중단됩니다.
- **장외 거래 차단** — 장 시간이 아닐 때는 매매하지 않습니다.

## 6. 자주 묻는 질문

**Q. 컴퓨터를 24시간 켜둬야 하나요?**
A. 아침(장 시작 전)에 켜두고 저녁에는 꺼도 됩니다.
   자동 종료 시각 설정이 있어서 장 마감 후 알아서 꺼집니다.

**Q. 돈을 얼마나 잃을 수 있나요?**
A. 투자이므로 손실 가능성은 있습니다.
   일일 손실 한도가 설정되어 있어서 그 이상은 자동으로 중지됩니다.

**Q. 잘못된 매매가 되면 어떻게 멈추나요?**
A. 텔레그램에 `/emergency_stop` 입력 → 즉시 모든 매매 중단.
   더 확실하게는 `/close_all` → 전량 청산(승인 필요).

**Q. AI 호출 비용이 많이 나오지 않나요?**
A. 하루 비용 한도가 설정되어 있습니다.
   `/cost`로 현재까지 쓴 금액을 확인할 수 있습니다.
